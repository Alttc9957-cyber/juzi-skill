> 从"凭感觉进化"到"有体系进化"

## 🚀 v2.2 核心升级：强制执行机制

### 解决了什么问题？
- **v2.1 问题**：AI 嘴上说"记住了""我会了"，实际上没有写入文件，进化体系"建而不用"
- **v2.2 方案**：新增强制执行模块，4个硬性触发器+自检清单，确保"做了"而不是"说了"

### 核心原则
```
做 > 说
写了 > 记住了  
证明了 > 承诺了
没有写入文件 = 没有发生
```

### 4个硬性触发器（不可跳过）
- 🔴 触发器A：用户说"不对" → 同一次回复写入 feedback 文件
- 🔴 触发器B：用户说"记住" → 同一次回复写入记忆文件
- 🔴 触发器C：工具选择前 → 必须先读 TOOLS.md
- 🔴 触发器D：写作任务开始 → 必须先读语料库/风格偏好

## 📖 快速开始

### 方式一：自举安装（推荐）

```
告诉你的 AI：
"读取 AI-Agent进化工具包/00_自举安装指南/自举安装_v2.2.md 并执行安装"
```

AI 会自动完成所有安装步骤。

### 方式二：手动喂食（v2.0 方式）

按以下顺序逐篇喂食：
1. 01_进化体系/进化机制_v2.0.md
2. 02_记忆体系/记忆方法论_v2.0.md
3. 03_深度调研SOP/调研管线_v2.0.md
4. 04_知识库体系/知识库架构_v2.0.md
5. 05_能力路由框架/路由决策树_v2.0.md
6. 06_思维工具箱/思维框架_v2.0.md
