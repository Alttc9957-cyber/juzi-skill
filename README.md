# AI Agent 进化工具包 v2.1

> 让 AI 智能体越用越好用的系统框架  
> 从"凭感觉进化"到"有体系进化"

## 🚀 v2.1 核心升级：自举安装

### 解决了什么问题？
- **v2.0 问题**：需要用户按顺序逐篇喂食文件，AI 不会主动内化
- **v2.1 方案**：AI 读取工具包后，自动将核心规则写入系统记忆，成为永久行为规则

### 怎么用？
```
1. 将整个工具包文件夹发给 AI
2. 告诉它："读取 00_自举安装指南/自举安装_v2.1.md 并执行安装"
3. AI 自动完成：读取 → 提取规则 → 写入记忆 → 创建目录 → 验证
4. 安装完成后，进化机制自动运行，无需手动提醒
```

### 效果对比

| 维度 | v2.0 | v2.1 |
|------|------|------|
| 安装方式 | 手动逐篇喂食 | 自动读取+自举安装 |
| 规则存储 | 外部文档 | MEMORY.md永久规则 |
| 触发方式 | 需要提醒 | 自动触发 |
| 持续性 | 每次会话需重新加载 | 一次安装，永久生效 |

### 核心能力

**进化体系 6 大机制**：
1. 复盘升级（三角验证法）
2. 反馈记忆（四大分类）
3. 知识库进化（三问闸门）
4. 学习机制（项目驱动）
5. 工具进化（3次法则）
6. 进化可观测（三指标）

**记忆体系 3 层架构**：
- 即时层（核心规则，自动加载）
- 近中期层（项目进度，索引检索）
- 长期层（全量历史，语义搜索）

## 📖 快速开始

### 方式一：自举安装（推荐）

```
告诉你的 AI：
"读取 AI-Agent进化工具包/00_自举安装指南/自举安装_v2.1.md 并执行安装"
```

AI 会自动完成所有安装步骤。

### 方式二：手动喂食（v2.0 方式）

按以下顺序逐篇喂食：
1. 01_进化体系/进化机制_v2.0.md
2. 02_记忆体系/记忆方法论_v2.0.md
3. 03_深度调研SOP/调研管线_v2.0.md
4. 04_知识库体系/知识库架构_v2.0.md
5. 05_能力路由框架/路由决策树_v2.0.md
6. 06_思维工具箱/思维框架_v2.0.md
