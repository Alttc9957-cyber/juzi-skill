# AI-Agent 进化工具包 v2.0

> 一套经过实战验证的 AI Agent 构建方法论与工具集。不卖焦虑，只给干货。

## 这是什么？

这是我们团队在企业 AI 落地过程中，踩了无数坑之后沉淀下来的一套完整体系。

从 Agent 怎么进化、记忆怎么设计、知识库怎么搭、能力怎么路由——到具体用什么思维框架去拆解问题，全在这里。

不是理论，是我们自己真在用的东西。

## 包含什么？

| 模块 | 内容 | 解决什么问题 |
|------|------|------------|
| `00_总控` | Agent 总控端设计 | 怎么让 Agent 像"项目经理"一样统筹全局 |
| `01_进化体系` | 进化机制与方法论 | Agent 怎么从"能用"变成"好用"，持续迭代 |
| `02_记忆体系` | 记忆架构设计 | 长期记忆、短期记忆、工作记忆怎么分层管理 |
| `03_深度调研SOP` | 调研管线流程 | 怎么做深度信息调研，不遗漏、不跑偏 |
| `04_知识库体系` | 知识库架构方案 | RAG 不是万能的，知识库该怎么分层、怎么准入 |
| `05_能力路由框架` | 路由决策树 | Agent 面对复杂任务，怎么判断该调什么工具、走什么路径 |
| `06_思维工具箱` | 思维框架合集 | 第一性原理、费曼学习法、批判性思维……实战中最好用的几个 |
| `07_使用指南` | 快速上手文档 | 拿到手就能用，不用看论文 |

## 适合谁？

- 在折腾 AI Agent、想让它真正好用的人
- 做企业 AI 落地、需要方法论参考的人
- 对 Agent 架构设计感兴趣、想少走弯路的人

不适合：想靠 AI 自动赚钱、不愿意动手的人。

## 怎么用？

1. 先看 `07_使用指南/快速上手_v2.0.md`，了解整体架构
2. 根据你的需求，挑对应模块深入看
3. 直接拿来改，用到你的项目里

每个模块都是独立的 markdown 文件，可以直接阅读、复制、修改。

## 关于差九年 AI 进化俱乐部

这套工具包来自「**差九年 AI 进化俱乐部**」。

### 为什么叫"差九年"？

我们的合伙人之间差九岁。差一点圆满，但一直在死磕。

我们做了一档播客叫「差九年」，聊的都是 AI 实战——不是那种"三个步骤让你月入十万"的扯淡，是真正在落地过程中踩的坑、踩过的雷、跑通的流程。

### 俱乐部在做什么？

- 📦 **持续迭代 AI 工具包**——这个 v2.0 只是起点，后续会持续更新
